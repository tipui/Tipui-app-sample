\Tipui\Builtin\HTML\Libs\FileSystem\SetCHMOD{}


Setting the permissions of files and folders

Only for Linux or systems that supports PHP chmod() function.

Parameters

(string) $path Full path of file to read.
(int:0777) $mode Linux file system permission code.

Returns

(bool)true or 1: Successful, however errors may happen regardless of successful response.
(int)2: Source path not exists.
