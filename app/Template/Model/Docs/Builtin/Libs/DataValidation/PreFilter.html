\Tipui\Builtin\Libs\DataValidation\PreFilter{}


Perform filters before proceed the execution of basic and customized rules.

PreFilter() is called when,
- The rule parameter pre_filter is defined.
- Auto load the Upload filter for data rule whose parameter type was declared as file.
