\Tipui\Builtin\Libs\Cookie\Set()


Setting cookie key


Custom cookie settings

(string) $key Array key name.
(id) $value Value for the key.

[Optional parameters]
(string) $time Amount of time (ie: +30, +12, -90)
Default: \Tipui\Core::GetConf() -> COOKIES -> COOKIE_TIME

(string) $time_mode Type of time. See PHP strtotime() function.
Default: \Tipui\Core::GetConf() -> COOKIES -> COOKIE_TIME_MODE

(string) $path URL bas path to apply cookie data.
Default: \Tipui\Core::GetConf() -> BOOTSTRAP -> PUBLIC_FOLDER

(string) $domain Targeted full qualified domain name.
Default: \Tipui\Core::GetConf() -> BOOTSTRAP -> DOMAIN

(string) $subdomain Subdomain name.
Default: \Tipui\Core::GetConf() -> BOOTSTRAP -> SUBDOMAIN
