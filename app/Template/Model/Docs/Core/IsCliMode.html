\Tipui\Core->IsCliMode

Check request cames from CLI (Command Line Interface) or HTTP protocol.
If PHP function php_sapi_name() returned string is cli, the Core property sapi_is_cli will be assigned with boolean TRUE. If not, will assigns FALSE.

Cached Core methods

For read the value, call the GetConf() method, that will retrieves data from the cache, see:
\Tipui\Core::GetConf()->GetMethodDataCache()
