\Tipui\Core::GetConf()


Returns new instance of Core object.
The properties bellow are virtualy created from magic methods into Core.
The magic method __get() identifies who is the parent method calling by checking the property Core -> called_from.
If Core->called_from is equal to Core::CONF_CALLER, then, returns the Core->GetEnv() array indexes.
