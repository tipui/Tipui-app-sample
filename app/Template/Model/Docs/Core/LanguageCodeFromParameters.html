\Tipui\Core->LanguageCodeFromParameters

Returns the language code stored in the user cache (session, cookie).
If language is not defined by the user, returns boolean FALSE.
Otherwise, returns the language code based on ISO 639-1.

Cached Core methods

For read the value, call the Core::GetConf() method, that will retrieves data from the cache:
\Tipui\Core::GetConf()->GetMethodDataCache('LanguageCodeFromParameters')
