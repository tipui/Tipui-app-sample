Tipui Namespaces


Rules

The Tipui namespaces nomenclatures base are as follows:
Core: Tipui\Builtin
App: Tipui\App
The full namespace nomenclature must be exact same as the name of path where the class is located.
The names follows the CamelCase pattern for compound words. For single words, must start with uppercase letter.
