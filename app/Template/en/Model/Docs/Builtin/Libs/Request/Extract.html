Extracting parameters

Responsible for extracting parameters from URL over HTTP or CLI mode.

HTTP: http://foo.bar/?p=bar&id=x
HTTP Mode Rewrite (Friendly URL): http://foo.bar/bar/x
CLI: /path/of/binary/php /path/of/app/public/index.php -p p=Bar%26id=x
CLI (Friendly): /path/of/binary/php /path/of/app/public/index.php -p Bar/x
