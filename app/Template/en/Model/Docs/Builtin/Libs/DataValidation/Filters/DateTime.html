\Tipui\Builtin\Libs\DataValidation\Filters\DateTime{}


Check if string is an valid date and time.
Pattern: ISO 8601

Can be used to perform sanitization also.
For example, string containing the time without the seconds 2014-01-24 17:18, will be converted to 2014-01-24 17:18:00.
