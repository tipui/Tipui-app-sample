\Tipui\Builtin\HTML\Libs\FileSystem\MoveUploadedFile{}


Moving uploaded files

Only for Linux or systems that supports PHP chmod() function.

Parameters

(string) $from Source path. The temporary path where the file was uploaded.
(string) $to Destiny path to move the file.

Returns boolean

(bool)true: Successful.
(bool)false: Couldn't be moved.
