\Tipui\Builtin\HTML\Libs\FileSystem\WriteFile{}


Writes / Puts content for a file

Writes content for existing file or creates new file if already exists.
Optional parameter $mode sets the chmod under Linux like operational systems.

Parameters

(string) $path Full path of file.
(string:[empty]) $content The content.
(bool:true) $overWrite Sets true for allow overwrite existing file, false for not overwrite.
(string:w+) $method Defines the type of access. See PHP fopen() function page.
(int:0777) $mode Linux file system permission code.

Returns

(bool)true or 1: Successful, however errors may happen regardless of successful response.
(int)2: file path not exists.
(int)3: The file couldn't be overwriten.
(int)4: The file couldn't be oppened.
(int)5: The file couldn't be written.
