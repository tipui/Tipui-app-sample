\Tipui\Core->__construct

Call the PHP debug_backtrace() function to abstracts who is calling the curretn Core instance.
Resets the property Core->called_getenv that holds the array for Core->GetEnv() parameters, to be invoked in chained style.
Assigns the Core->called_from with name of method that is instantiating the Core.
